\section{Objectius Assolits}

Hem de remarcar que tot i que el sistema està destinat a l'extracció de referències, la part de generació de regles i la manera d'aplicar-les realment és la base d'un \textit{framework} que es podria utilitzar per qualsevol altre sistema similar. En cas de no tractar-se de documents HTML, només caldria afegir els nous tipus de regles necessaris.
